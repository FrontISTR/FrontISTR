# 現状の実装状況

## 実装済み

* MITC4シェル要素に対して、幾何学的非線形解析（`NLGEOM`）用の処理を追加した。
* 主な対象は、要素タイプ `741`、4節点、Total Lagrangian、弾性材料の組み合わせである。
* シェル節点の有限回転状態を、節点ごとのdirectorおよびtriadとして保持するようにした。
* Newton反復中に、線形ソルバから得られた回転増分を用いてシェル節点の試行姿勢を更新し、収束時にその状態を確定する処理を追加した。
* 接線剛性行列の計算では、参照配置のdirectorと現在の試行配置のdirectorを用いるようにした。
* 内部力、応力、ひずみの評価において、Total Lagrangianに基づく有限変形の処理を追加した。
* 応力状態に基づく幾何剛性項を、接線剛性行列に加える処理を追加した。
* 回転境界条件では、現在の回転状態から目標回転状態への相対回転を求め、それを増分回転として扱う処理を追加した。
* cutback発生時に、シェル節点の回転状態と厚み方向Gauss点履歴を保存・復元する処理を追加した。
* 節点応力出力では、シェル厚み方向Gauss点に保持された応力・ひずみ履歴を利用する処理を追加した。

## 今回の対象外

* 要素タイプ `731` および `743` は、今回の対応対象外である。
* mixed shell-solid 系の要素は、今回の対応対象外である。
* 塑性、クリープ、粘弾性などの非弾性材料への対応は、今回の対応対象外である。
* 動解析への拡張は、今回の対応対象外である。

## 確認状況

* MR用ソースツリーから正常にビルドできることを確認した。
* 要素タイプ `741` のシェル要素を用いた `NLGEOM` 静解析ケースで、解析が正常終了することを確認した。
